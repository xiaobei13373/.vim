### 基础操作
| 快捷键 | 功能说明 |
|--------|----------|
| `S` | 保存文件 (`:w`) |
| `Q` | 退出 Vim (`:q`) |
| `R` | 重新加载 vimrc 配置 (`:source $MYVIMRC`) |
| `;` | 进入命令模式 (`:`) |
| `<Leader><CR>` | 清除搜索高亮 (`:nohlsearch`) |

| 快捷键 | 功能说明 |

## 主要配置特性

- **行号显示**: 显示绝对行号
- **光标行高亮**: 当前行高亮显示
- **搜索高亮**: 搜索时高亮匹配项，按 `<Leader><CR>` 清除
- **智能大小写**: 搜索时根据输入自动判断是否区分大小写
- **缩进设置**: 4 空格缩进，使用空格代替 Tab
- **鼠标支持**: 启用鼠标操作
- **自动切换目录**: 打开文件时自动切换到文件所在目录
- **记忆光标位置**: 重新打开文件时恢复到上次编辑位置
- **光标形状变化**: 插入/替换/正常模式下光标形状不同
